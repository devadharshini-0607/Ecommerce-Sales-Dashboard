# E-Commerce Sales & Profit Analysis Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze e-commerce sales performance, profitability, customer behavior, and payment trends.

## Tools Used
- Power BI
- Excel

## Dataset
- Orders Dataset: 1500+ records
- Details Dataset: 500+ records

## Dashboard Features
- Sales Analysis
- Profit Analysis
- Customer Analysis
- State-wise Analysis
- Payment Mode Analysis
- Monthly Profit Trends

## Key Metrics
- Total Profit: 37K
- Total Amount: 438K
- Total Quantity: 5615
- Average Order Value: 121K
